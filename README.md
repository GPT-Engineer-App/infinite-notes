# infinite-notes

A note-taking app with:

- Notebooks (pages of infinite length)
- Markdown support
- Data saved on-device, autosave ensures your notes are always there
- The ability to lock (password protect) notebooks

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/infinite-notes.git
cd infinite-notes
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
